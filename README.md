# WPModernUI

A WordPress plugin that modernizes admin pages with clean, tabbed interfaces.

## Plugin Location

The installable plugin is in the `modern-admin-ui/` folder.

## Quick Install

1. Copy the `modern-admin-ui/` folder to your WordPress `/wp-content/plugins/` directory
2. Activate "Modern Admin UI" in WordPress Admin > Plugins
3. Go to "Modern Admin UI" in the admin menu
4. Enable the pages you want to modernize
